# Рабочее время DK

Локальное PWA-приложение для учёта рабочих смен, часов и примерной зарплаты в Дании.

## Что умеет

- календарь по месяцам;
- смены с началом, концом и перерывом;
- почасовая ставка и доплаты: вечер, ночь, выходной, overtime, lokal-/особая надбавка;
- итоги месяца: часы, брутто, feriepenge, примерная сумма после AM-bidrag и A-skat;
- сохранение данных в браузере через localStorage;
- manifest и service worker для установки на телефон.

## Как открыть

Для обычной проверки можно запустить локальный сервер в этой папке и открыть:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Адрес:

```text
http://127.0.0.1:4173/
```

Чтобы установить на телефон, страницу нужно открыть с телефона через HTTPS или через локальную сеть с рабочим PWA-хостингом. После этого в Chrome/Android появится установка приложения, а в Safari/iPhone можно выбрать "Share" -> "Add to Home Screen".

## Важно

Это личный расчётчик, а не официальный датский расчёт зарплаты. Налог зависит от налоговой карты, fradrag, коммуны, пенсии, ATP, договора и overenskomst. Все проценты и доплаты можно менять в настройках.
